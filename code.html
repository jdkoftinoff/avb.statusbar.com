jdksavdecc-c

Description:
A library for the AVDECC protocol in the C programming language.
Repository:
https://github.com/jdkoftinoff/jdksavdecc-c
git clone git@github.com:jdkoftinoff/jdksavdecc-c.git
Documentation:
http://avb.statusbar.com/docs/jdksavdecc-c/
Prerequisites:
None

Obbligato

Description:
A musical line that is in some way indispensable in performance. A foundation library for the C++11 programming language.
Repository:
https://github.com/jdkoftinoff/Obbligato
git clone git@github.com:jdkoftinoff/Obbligato.git
Documentation:
http://avb.statusbar.com/docs/Obbligato/
Prerequisites:
A C++11 compiler (either Clang 3.1+ or GCC 4.7+ or Microsoft Visual Studio 2012)

Stretto

Description:
A concluding passage in a composition, played at a faster speed than the earlier material. Examples and utilities for AVB in the C++11 programming language.
Repository:
https://github.com/jdkoftinoff/Stretto
git clone git@github.com:jdkoftinoff/Stretto.git
Documentation:
http://avb.statusbar.com/docs/Stretto/
Prerequisites:
A C++11 compiler (either Clang 3.1+ or GCC 4.7+ or Microsoft Visual Studio 2012), Obbligato, and jdksavdecc-cpp

jdksavdecc-proxy

Description:
an open source implementation of the IEEE 1722.1 Layer 3 Proxy Server in implemented for posix targets in minimal C programming language.
Repository:
https://github.com/jdkoftinoff/jdksavdecc-proxy
git clone git@github.com:jdkoftinoff/jdksavdecc-proxy.git
Documentation:
http://avb.statusbar.com/docs/jdksavdecc-proxy/
Prerequisites:
Posix platform, C Compiler, and microsupport
